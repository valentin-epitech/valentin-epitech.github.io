This game is informally called a Horizontal Shmup (e.g. Shoot'em'up), and while R-Type is not the first one of its category, this one has been a huge success amongst gamers in the 90's, and had several ports, spin-offs, and 3D remakes on modern systems.

This project is a project for our third year at EPITECH, the goal: Implement a multithreaded server and a graphical client for a game called R-Type, using an engine of your own design.

Although what we did isn't pixel perfect, we're all proud of what we did !
